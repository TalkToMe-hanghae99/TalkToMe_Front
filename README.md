##❤ 톡투미(TalkToMe)소개
아주 사소한 것부터 진지한 것까지 머릿속으로 생각만 하던 고민들을  털어놓을 수 있는 공간입니다. 
사람들의 고민에 공감을 할 수도, 답변을 달아 줄 수도 있고 선택의 기로에 놓였을 때 다른 사람들의 의견을 들어보고 선택의 폭을 좁힐 수 있습니다.

## 🙋 팀원

- Front-end(React): 박새봄, 신유빈, 조민갑
- Back-end(Node.js): 오정민, 홍성현
